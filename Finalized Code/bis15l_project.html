Introduction

Our project describes the spread of C. auris, an emerging fungal pathogen, within the United States since 2012 and visualizes known associations between the pathogen’s antimicrobial resistance (AMR) genes and drug resistance. Our shared interest in epidemiology and AMR motivated us to explore this topic.

C. auris became nationally notifiable in the United States in 2018, but it is currently not reportable in all states. Nationally notifiable conditions are reported voluntarily by states where the condition is reportable, so not all outbreaks of C. auris are captured in the CDC’s National Notifiable Disease Surveillance System. Our main objective was to describe the spread of C. auris in the United States over time using surveillance data from the CDC. We were also interested in determining whether Google Trends data on the frequency of Google searches related to c. auris could serve as an indicator for locations of outbreaks that are not reported to the CDC. Lastly, to describe the degree to which C. auris may pose a threat to public health in the United States, we explored how exisiting genetic variation in C. auris may contribute to antifungal resistance and severity of infection using data from Chow et al 2020 [5] and the NCBI Pathogen Detection Isolates Browser.

Conclusions

Using surveillance and genetic data, we visualized the spread of C. auris in the United States and analyzed associations between genetic factors with both antifunagal resistance and severity of infection. Since the incidence and the frequency of drug resistance is increasing, C. auris is an emerging public health threat that should be monitored by public health professionals. Google Trends may serve as a useful monitoring tool for C. auris because it can help identify outbreaks that have not been captured in the current voluntary reporting system.

# Separate top queries from rising queries.
top_related <- related %>%

  rename(query = TOP) %>% 
  slice(-1) %>% 
  mutate(query = str_remove(query, ",Breakout"))

## Obtain counts for words in top queries:
# Create Separate columns for counts and queries.
top_counts <- top_related %>% 
  separate(TOP, c("query", "count"), ",") %>% 
  head(-2) # remove candida and auris

# Use text mining to get total counts for each word.
# Create and preprocess a corpus for top queries.
top_corpus <- Corpus(VectorSource(top_counts$query)) %>% 
  tm_map(removeWords, c("is", "of", "candida", "auris"))

# Create a document term matrix from the corpus.
# Each document (query) is represented by a set of tokens (words) and their counts.
top_dtm <- TermDocumentMatrix(top_corpus) %>% 
  as.matrix()

# Get word frequencies from the dtm.
top_words <- rowSums(top_dtm) 
top_totals <- tibble(word = names(top_words), freq = top_words) 
# freq > 1 = duplicate words
# multiply freq by count for each word

# Combine word frequencies with query counts to get totals for each word.
top_totals <- top_totals %>% 
  mutate(count = c((100+14), 27, (25+8), 20, 19, 17, 17, 
                   16, 14, (12+10), 10, 9, 8, 8)) %>% 
  mutate(n = freq*count)

## Create the word cloud.
set.seed(1)
ggplot(data = top_totals) +
  geom_text_wordcloud(aes(label = word , size = n, color = word)) +
  scale_size_area(max_size = 28) +
  scale_color_manual(values = paletteer_c("grDevices::Teal", 14)) +
  theme_minimal()

                          col_names = TRUE) %>% 
  rename(region = Region,
         count = `Candida auris: (3/9/21 - 3/9/22)`)

# Add the abbreviation column for faster filtering.
searches_2021 <- inner_join(searches_2021, states, by = "region") %>% 
  inner_join(state_key, by = "region") %>% 
  rename(abb = jurisdiction) %>% 
  relocate(abb, .before = count)

# Relate the center coordinates to the number of searches in each state.
searches_2021_coords <- inner_join(searches_2021 %>% 
                                     select(region, abb, count),
                                   center_coords,
                                   by = "region")

# Create a vector containing the abbreviations of states of interest.
# Includes states with high number of reported cases or google searches.
states_of_interest <- c("CA", "IL", "NY", "FL", "OR")
soi_2021_coords <- searches_2021_coords %>% 
  filter(abb %in% states_of_interest)

ggplot(data = searches_2021) +
  geom_polygon(aes(x = long, y = lat, group = group, fill = count)) +
  geom_label(data = soi_2021_coords,
             aes(x = long, y = lat, label = count),
             fontface = "bold") +
  theme_void() +
  scale_fill_viridis_c(option = "mako", direction = -1, na.value = "gray") +
  labs(fill = "Count",
       title = 'Number of Google Searches Containing "Candida auris" in 2021') +
  theme(plot.title = element_text(size = 14,
                                  margin = margin(t = 15, b = -15)),
        legend.margin = margin(10, 30, 10, 10))
